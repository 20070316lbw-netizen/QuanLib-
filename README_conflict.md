# QuanLib 量化代码素材库

## 功能说明
QuanLib 是一个面向研究与实盘的**只读代码素材库**。这里的每个模块都是独立可运行的代码示例，均自带测试数据和使用说明，用于快速复制、修改并集成到您的新项目中。

**核心原则:**
- ✅ 每个模块都自带 example 和测试数据
- ✅ 代码简洁，注释完整，可直接复制使用
- ✅ 此库为代码组件仓库，非完整项目工程

## 模块总览

| 模块目录 | 功能描述 | 核心内容 |
| --- | --- | --- |
| `01_data_quality/` | 数据质量检查模块 | 提供OHLCV完整性检查、价格条查等通用检查器 |
| `02_neutralization/` | 因子中性化模块 | 行业中性化、市值中性化、OLS 矩阵处理封装 |
| `03_models/` | 手写模型库 | 包含底层自研算法如 `SimpleGBDT`、OLS线性回归等 |
| `04_factors/` | 因子计算库 | 动量因子 (12-1)、反转因子、波动率、估值因子 (BP/EP) 等 |
| `05_evaluation/` | 回测与评估指标 | 指标计算 (IC, RankIC, ICIR) 及简易向量化回测框架 |
| `06_utils/` | 工具函数 | 交易日历处理、股票状态过滤 (ST/停牌)、日志 (loguru) 模板 |

## 快速开始

1. 确保安装了基础依赖（见 `requirements.txt`）
2. 进入你需要参考的模块，例如 `cd 01_data_quality`
3. 运行对应的演示脚本，例如 `python example_usage.py`
4. 查看输出结果。如果您觉得该组件有用，可直接将模块下的核心 Python 文件复制到您的策略代码中。
